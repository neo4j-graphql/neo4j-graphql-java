= Plural directive

== Setup

.Schema
[source,graphql,schema=true]
----
type Tech @plural(value: "Techs") {
  name: String
}
----

== Select Tech with plural techs

.GraphQL-Query
[source,graphql]
----
{
  techs {
    name
  }
}
----

.Expected Cypher params
[source,json]
----
{ }
----

.Expected Cypher output
[source,cypher]
----
MATCH (this:Tech)
RETURN this {
	.name
} AS this
----

'''
