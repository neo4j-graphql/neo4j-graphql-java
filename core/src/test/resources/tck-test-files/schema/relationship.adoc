= Schema Relationship

Tests that the provided typeDefs return the correct schema (with relationships).

== Inputs

[source,graphql,schema=true]
----
type Actor {
    name: String
}

type Movie {
    id: ID
    actors: [Actor]! @relation(name: "ACTED_IN", direction: IN)
}
----
