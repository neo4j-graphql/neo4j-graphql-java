= Schema Simple

Tests that the provided typeDefs return the correct schema.

== Setup

.Schema
[source,graphql,schema=true]
----
type Movie {
    id: ID
    actorCount: Int
    averageRating: Float
    isActive: Boolean
}
----

.Configuration
[source,json,schema-config=true]
----
{
  "queryOptionStyle": "INPUT_TYPE",
  "useWhereFilter": true,
  "pluralizeFields": true
}
----
